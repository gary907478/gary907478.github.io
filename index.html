How to setup local WordPress env

Posted on 2019-11-09

##How to Setup Local WordPress Env

##MAMP

  1. Install MAMP from https://www.mamp.info/en/
  2. get Apache port number. (preference -> ports)
  3. get Document Root. (preference -> web-server)
  4. Open Webstart page
  5. go to phpMyadmin (in Tools)
  6. click new on left sidebar. Create database name.
  7. Drag your WP file into DOC root folder (normally htdocs)
  8. go to your local apache port url
  9. get start!
  • Some problems and solution
    • Apache server does not work.
      • try different PHP version in MAMP setting.

##WP
Generally, When you drag clean new WP file into htdocs. Next time, when you open localhost, it will show you a link. Click the link will lead you to install WP.

PostgreSQL Notes

Posted on 2019-05-14

installation

  • brew update before installation.
  • brew install postgresql
  • download PSequel

Command

creat database

  • creatdb ‘dbname’

creat table

  • CREATE TABLE table_name (dataName dataType, dataName dataType);
    • ex. CREATE TABLE users (name text, age smallint, birthday date);

insert data into table

  • INSERT INTO table_name(dataName1, dataName2, dataName3) VALUES (value1, value2, value3);
    • ex. INSERT INTO users(name, age, birthday) VALUES (‘Gary‘, 25, 1994-01-01);

check datas

  • SELECT dataName1,dataName2,dataName3 FROM table_name;
    • ex. SELECT name, age, birthday FROM users;

check datas with condition

  • SELECT * FROM users WHERE name LIKE ‘A%’;

    • search all datas that name start with A.

  • SELECT * FROM users ORDER BY weight DESC/ASC;

    • select all datas with descending/ascending order. 降序/升序

alter table

  • ALTER TABLE table_name ADD dataName dataType;
    • ex. ALTER TABLE users ADD weight smallint;

update datas

  • UPDATE table_name SET dataName = value WHERE dataName = value;
    • UPDATE users SET weight = 60 WHERE name = ‘Gary’;

functions

  • AVG() calculate average of datas
  • SUM() calculate sum of datas
  • COUNT() count numbers of datas

Tai Lopez的四条快速阅读原则

Posted on 2019-03-14

首先，精选书籍。 洛佩兹说，要区分一般人和亿万富翁的差别，就是看他们读什么，他建议人们应花时间去阅读知名作者所创作的、有品质的书，比如沃尔玛创始人山姆.沃尔顿总结自己毕生所学的自传。（我个人觉得吴晓波老师写的书都可以阅读学习）

第二，不同的时间读不同的书。洛佩兹建议，在不同的时间段，阅读内容应有所区别，早餐阅读那些出版时间超过50年的经典作品，比如弗洛伊德、柏拉图，让经典激发你的大脑；白天时间则可以阅读工具书，或者是和工作相关的文章，精进工作技能；晚上可以读名人自传，激发斗志。（我觉得什么时间段读什么书不必一定按照洛佩兹所说的来，可以根据自己的习惯和环境来尝试不同的阅读节奏，最终养成适合自己的阅读习惯）

第三，找各种时间阅读。知道该读什么之后，就要想办法读完。亿万富翁之所以看书很快，是因为他们想要获得只是、主动寻找任何可以阅读的时间。洛佩兹认为，每个人应该把一天可以利用时间平均分成三份，三分之一用来阅读，三分之一用来消化知识；最后三分之一执行自己所学到的知识。（我觉得这里面有洛佩兹自己的经历来支撑他的说法，不过这种三分法却是可以落实一下试试，不过不知道有多少人可以做到这种状态，学以致用。）

第四，投入大量时间和金钱阅读。洛佩兹说，有两种行为不会让你破产，一是从事慈善，二是投资知识，学的越多，得到的越多。因此不妨像亿万富翁一样，花大量时间和金钱投资知识。（个人觉得功利的来学习知识是有些局限性的，虽然说有些知识会在特殊场合中变得无用，但是知识的积累确实应该作为一种习惯来培养的，社会的分工会越来越细，但是如果只守着自己的“舒适区“，那就只能在竞争不激烈的环境中慢慢退步了。）

《为什么精英都是时间控》读书笔记

Posted on 2019-02-28 | Edited on 2019-03-01 | In 读书笔记

《为什么精英都是时间控》

书名： 为什么精英都是时间控
作者： 桦泽紫苑 [日]
日期： July 19, 2018.

笔记



  • 专注性工作放在上午做，上午的时间价值是下午晚上的四倍。
  • 掌握工作节奏，不要牺牲睡眠，这样反倒低效。 利用有氧运动给大脑充能。
  • 提高工作效率。自由时间不要国度投入工作。 利用自由时间投资自己进行主动性娱乐（读书，健身等有益于自身的娱乐。减少玩游戏，看电视等低效娱乐）
  • 提高专注力
    • 15·45·90法则。 以15分钟为一个单位进行定时专注工作。（类似番茄钟）
    • 排除杂念
      • 整理工作台，越简洁越不会分心。
      • 想到的事情立马记录下来，然后考虑是否去做。
      • 减少通讯带来的杂念。（关闭手机，电脑上的通讯软件等）专注于手头上的工作。
      • 给自己的工作设置deadline。绝对不能有拖延的情况。
  • 早起，冲澡，吃早饭。
  • 午睡 30mins
  • 晚睡前不要吃东西，看电子产品。
  • 周末尽量不要睡懒觉，保持良好的睡眠习惯。

总结

和其他同类型效率工具书一样，读起来感觉都是干货。但是又觉得只是把网络上大家都知道的东西重新说了一遍。
